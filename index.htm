assume.js is a simple jQuery algorithm that uncomplicates the process of calculating the correct number of shares when co-founding a start-up.

When you want to start a project with a team and it's time to start a company, it's wise to be realistic and try to objectively identify how much value and responsibility each member can start with.

I regard the following factors as the most important to take into the estimation (this algorithm will be improved over time):

What type of entity naturally brings more weight/value in this project?

Administrator?

Manifester?
